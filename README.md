# Hello!

this is a project that uses nextjs to convert words to semi-numbers
(also called [leet speak](<(https://en.wikipedia.org/wiki/Leet)>))

## for example:

input:

Hello World!

output:

HƐllØ WØгld!

### OR:

input:

ELEET HAXOR

output:

ƐLƐƐT H4×Øг
